About Quit

"Quit" a shutdown manager for Linux terminal. It is developed in C++.

Features

Quit stops your computer from the command line. It has four opcions:

  1. Reboot
  2. Suspend
  3. Shutdown
  4. Timed shutdown

Requirements

You don't need practically anything to install Quit,
just a computer with Linux.

Installation

You can install Quit from Snapstore.

But you can also install it from the portable relase
